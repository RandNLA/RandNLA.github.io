Subspace Embedding via Leverage Score Sampling

On this page, we show how sampling a matrix via its leverage scores creates a subspace embedding guarantee. Subspace embeddings are core to much of RandNLA, so many papers either assume they are given a subspace embedding, or create a proof much like the one below (just fine-tuned to research their setting).

Prerequisite: Basic Properties of Leverage Scores

We consider the following algorithm:

Algorithm 1: Leverage Score Subspace Embedding

input: Matrix A∈Rn×d\boldsymbol{A}\in\mathbb{R}^{n \times d}A∈Rn×d. Number kkk of subsamples.

output: Sketched matrix A~∈Rk×d\tilde{\boldsymbol{A}}\in\mathbb{R}^{k \times d}A~∈Rk×d

  1. Sample indices s1,…,sk∈[n]s_1,\ldots,s_k\in[n]s1​,…,sk​∈[n] iid with respect to the leverage scores of A\boldsymbol{A}A

  2. Let pi := τidp_i \;{\vcentcolon=}\; \frac{\tau_i}{d}pi​:=dτi​​ denote the probability of sampling row iii

  3. Build the sample-and-rescale matrix S∈Rk×n\boldsymbol{S}\in\mathbb{R}^{k \times n}S∈Rk×n:

    Row iii of S\boldsymbol{S}S has form [00⋯01kpsi0⋯0]\begin{bmatrix}0&0&\cdots&0&\frac{1}{\sqrt{k p_{s_i}}}&0&\cdots&0\end{bmatrix}[0​0​⋯​0​kpsi​​​1​​0​⋯​0​], where index sis_isi​ is the nonzero entry.

  4. Return A~=SA\tilde{\boldsymbol{A}} = \boldsymbol{S}\boldsymbol{A}A~=SA

This algorithm is equivalent to building A~\tilde{\boldsymbol{A}}A~ by sampling kkk rows of A\boldsymbol{A}A with respect to leverage scores, rescaling those rows by 1kpsi\frac{1}{\sqrt{k p_{s_i}}}kpsi​​​1​, and stacking those rows up. The matrix S\boldsymbol{S}S is just a formalization that makes it easier to analyze this algorithm.

Theorem 1: Subspace Embedding
Theorem

Fix ε>0\varepsilon > 0ε>0, δ>0\delta > 0δ>0 and run the above algorithm with k=O(dlog⁡(dδ)ε2)k = O(\frac{d\log(\frac d\delta)}{\varepsilon^2})k=O(ε2dlog(δd​)​). Then with probability 1−δ1-\delta1−δ, we have for all x∈Rd\mathbf{x}\in\mathbb{R}^{d}x∈Rd

(1−ε)∥Ax∥22≤∥SAx∥22≤(1+ε)∥Ax∥22 (1-\varepsilon)\|\boldsymbol{A}\mathbf{x}\|_2^2 \leq \|\boldsymbol{S}\boldsymbol{A}\mathbf{x}\|_2^2 \leq (1+\varepsilon)\|\boldsymbol{A}\mathbf{x}\|_2^2 (1−ε)∥Ax∥22​≤∥SAx∥22​≤(1+ε)∥Ax∥22​

To prove this result, we will first massage the Subspace Embedding Guarantee into a more approachable form. First, let U∈Rn×d\boldsymbol{U}\in\mathbb{R}^{n \times d}U∈Rn×d be any matrix with orthonormal columns with the same columnspace as A\boldsymbol{A}A (e.g. the U\boldsymbol{U}U matrix in the SVD of A\boldsymbol{A}A). Then, for any x∈Rd\mathbf{x}\in\mathbb{R}^dx∈Rd we have a one-to-one map to a vector y∈Rd\mathbf{y}\in\mathbb{R}^dy∈Rd such that Ax=Uy\boldsymbol{A}\mathbf{x}=\boldsymbol{U}\mathbf{y}Ax=Uy. So, we need to show that

(1−ε)∥Uy∥22≤∥SUy∥22≤(1+ε)∥Uy∥22∀y∈Rd\begin{aligned} (1-\varepsilon)\|\boldsymbol{U}\mathbf{y}\|_2^2 \leq \|\boldsymbol{S}\boldsymbol{U}\mathbf{y}\|_2^2 \leq (1+\varepsilon)\|\boldsymbol{U}\mathbf{y}\|_2^2 && \forall\mathbf{y}\in\mathbb{R}^d \end{aligned}(1−ε)∥Uy∥22​≤∥SUy∥22​≤(1+ε)∥Uy∥22​​​∀y∈Rd​

Next, we expand the ℓ2\ell_2ℓ2​ norms:

(1−ε)y⊺U⊺Uy≤y⊺U⊺S⊺SUy≤(1+ε)y⊺U⊺Uy∀y∈Rd\begin{aligned} (1-\varepsilon)\mathbf{y}^\intercal\boldsymbol{U}^\intercal\boldsymbol{U}\mathbf{y} \leq \mathbf{y}^\intercal\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{S}\boldsymbol{U}\mathbf{y} \leq (1+\varepsilon)\mathbf{y}^\intercal\boldsymbol{U}^\intercal\boldsymbol{U}\mathbf{y} && \forall\mathbf{y}\in\mathbb{R}^d \end{aligned}(1−ε)y⊺U⊺Uy≤y⊺U⊺S⊺SUy≤(1+ε)y⊺U⊺Uy​​∀y∈Rd​

Note that U⊺U=I\boldsymbol{U}^\intercal\boldsymbol{U}=\boldsymbol{I}U⊺U=I by definition, so we get

(1−ε)y⊺y≤y⊺U⊺S⊺SUy≤(1+ε)y⊺y∀y∈Rd\begin{aligned} (1-\varepsilon)\mathbf{y}^\intercal\mathbf{y} \leq \mathbf{y}^\intercal\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{S}\boldsymbol{U}\mathbf{y} \leq (1+\varepsilon)\mathbf{y}^\intercal\mathbf{y} && \forall\mathbf{y}\in\mathbb{R}^d \end{aligned}(1−ε)y⊺y≤y⊺U⊺S⊺SUy≤(1+ε)y⊺y​​∀y∈Rd​

For y=0\mathbf{y}=0y=0, the claim trivially holds, so assuming y≠0\mathbf{y}\neq0y=0 we divide by y⊺y\mathbf{y}^\intercal\mathbf{y}y⊺y on both sides:

(1−ε)≤y⊺U⊺S⊺SUyy⊺y≤(1+ε)∀y∈Rd\begin{aligned} (1-\varepsilon) \leq \frac{\mathbf{y}^\intercal\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{S}\boldsymbol{U}\mathbf{y}}{\mathbf{y}^\intercal\mathbf{y}} \leq (1+\varepsilon) && \forall\mathbf{y}\in\mathbb{R}^d \end{aligned}(1−ε)≤y⊺yy⊺U⊺S⊺SUy​≤(1+ε)​​∀y∈Rd​

By the Courant–Fischer-Weyl Min-Max Principle, this is equivalent to guaranteeing that

1−ε≤λi(U⊺S⊺SU)≤1+ε∀i∈[d]\begin{aligned} 1-\varepsilon \leq \lambda_i(\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{S}\boldsymbol{U}) \leq 1+\varepsilon && \forall i\in[d] \end{aligned}1−ε≤λi​(U⊺S⊺SU)≤1+ε​​∀i∈[d]​

And rearranging the claim, this becomes

∣λi(U⊺S⊺SU)−1∣≤ε∀i∈[d]\begin{aligned} |\lambda_i(\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{S}\boldsymbol{U}) - 1| \leq \varepsilon && \forall i\in[d] \end{aligned}∣λi​(U⊺S⊺SU)−1∣≤ε​​∀i∈[d]​

Or, equivalently,

∥U⊺S⊺SU−I∥2≤ε \|\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{S}\boldsymbol{U} - \boldsymbol{I}\|_2 \leq \varepsilon ∥U⊺S⊺SU−I∥2​≤ε

Note that this guarantee seems reasonable. We can verify that E[S⊺S]=I\mathbb{E}[\boldsymbol{S}^\intercal\boldsymbol{S}]=\boldsymbol{I}E[S⊺S]=I, so we have E[U⊺S⊺US]=U⊺U=I\mathbb{E}[\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{U}\boldsymbol{S}] = \boldsymbol{U}^\intercal\boldsymbol{U} = \boldsymbol{I}E[U⊺S⊺US]=U⊺U=I, and thus ∥E[U⊺S⊺SU]−I∥2=0\|\mathbb{E}[\boldsymbol{U}^\intercal\boldsymbol{S}^\intercal\boldsymbol{S}\boldsymbol{U}] - \boldsymbol{I}\|_2 = 0∥E[U⊺S⊺SU]−I∥2​=0. In other words, this guarantee looks like a concentration of a random variable around its mean.

We now verify this claim that E[S⊺S]=I\mathbb{E}[\boldsymbol{S}^\intercal\boldsymbol{S}]=\boldsymbol{I}E[S⊺S]=I. First, we let ri\mathbf{r}_iri​ denote row iii of S\boldsymbol{S}S:

ri=[00⋯01kpsi0⋯0] \mathbf{r}_i = \begin{bmatrix}0&0&\cdots&0&\frac{1}{\sqrt{k p_{s_i}}}&0&\cdots&0\end{bmatrix} ri​=[0​0​⋯​0​kpsi​​​1​​0​⋯​0​]

See Also

The proofs above are most closely related to their highly generalized analysis in Theorem 5 from Avron et al. (2019).

There are many extensions of leverage scores in the literature:

  • (Cohen Musco² (2017)) Ridge Leverage Scores, which relate to Ridge Regression.

  • (Alaoui Mahoney (2015)) Kernel Ridge Leverage Scores, which relate to Kernel Ridge Regression.

  • (Avron et al. (2017)) Kernel Ridge Leverage Function, which relates to Random Fourier Features for Kernel Ridge Regression.

  • (Chen Price (2019)) Leverage Function, which relates to Linear Operators instead of matrices. Sometimes called a Sensitvitiy Score.

  • (Avron et al. (2019)) Ridge Leverage Function, which relates to Linear Operators instead of matrices for Ridge Regression .

  • (Cohen Peng (2019)) Lewis Weights, which relate to LpL_pLp​ norms instead of just L2L_2L2​ norms.
