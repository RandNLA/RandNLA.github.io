The Johnson-Lindenstrauss Lemma

The Johnsons-Lindenstrauss Lemma, or JL Lemma, says that we can embed nnn points from Rd\mathbb{R}^dRd in a O(log⁡(n)ε2)O(\frac{\log(n)}{\varepsilon^2})O(ε2log(n)​) dimensional space while preserving all pairwise distances to relative error ε\varepsilonε. We prove this in the simplest case, where we choose this embedding to be a random Gaussian matrix. We start with a simple concentration for a single vector x∈Rd\mathbf{x}\in\mathbb{R}^{d}x∈Rd:

Lemma 1: Gaussians Preserve Norms
Lemma

Fix ε>0\varepsilon>0ε>0, δ>0\delta>0δ>0, dimension ddd, and k=Ω(1ε2log⁡(1δ))k = \Omega(\frac1{\varepsilon^2}\log(\frac1\delta))k=Ω(ε21​log(δ1​)). Let G∈Rk×d\boldsymbol{G}\in\mathbb{R}^{k \times d}G∈Rk×d be a matrix of iid N(0,1)\mathcal{N}(0,1)N(0,1) entries. Then with probability 1−δ1-\delta1−δ, for any fixed x∈Rd\mathbf{x}\in\mathbb{R}^dx∈Rd, we have

(1−ε)∥x∥22≤∥1kGx∥22≤(1+ε)∥x∥22 (1-\varepsilon)\|\mathbf{x}\|_2^2 \leq \|{\textstyle\frac{ 1}{ \sqrt k}}\boldsymbol{G}\mathbf{x}\|_2^2 \leq (1+\varepsilon)\|\mathbf{x}\|_2^2 (1−ε)∥x∥22​≤∥k​1​Gx∥22​≤(1+ε)∥x∥22​

That is, w\mathbf{w}w is just a vector of iid N(0,∥x∥22)\mathcal{N}(0,\|\mathbf{x}\|_2^2)N(0,∥x∥22​) random variables. So,

1k∥w∥22=1k∑i=1kwi2∼∥x∥22⋅1k∑i=1k(N(0,1))2 \frac1k\|\mathbf{w}\|_2^2 = \frac1k \sum_{i=1}^k w_i^2 \sim \|\mathbf{x}\|_2^2 \cdot \frac1k\sum_{i=1}^k(\mathcal{N}(0,1))^2 k1​∥w∥22​=k1​i=1∑k​wi2​∼∥x∥22​⋅k1​i=1∑k​(N(0,1))2

∣1k∑i=1kzi2−1∣≤8kln⁡(2δ)≤ε \left|\frac{1}{k}\sum_{i=1}^k z_i^2 - 1\right| \leq \sqrt{\frac{8}{k}\ln(\frac2\delta)} \leq \varepsilon ∣∣​k1​i=1∑k​zi2​−1∣∣​≤k8​ln(δ2​)​≤ε

where our value of k=Ω(1ε2log⁡(1δ))k = \Omega(\frac1{\varepsilon^2}\log(\frac1\delta))k=Ω(ε21​log(δ1​)) guarantees this error term is at most ε\varepsilonε. We conclude that

Given this simple concentration, we can state the Johnson-Lindenstrauss Lemma:

Lemma 2: Johnson-Lindenstrauss
Lemma

Let x1,…,xn∈Rd\mathbf{x}_1,\ldots,\mathbf{x}_n\in\mathbb{R}^{d}x1​,…,xn​∈Rd. Fix ε∈(0,1)\varepsilon\in(0,1)ε∈(0,1), δ>0\delta>0δ>0, and k=Ω(1ε2log⁡(nδ))k = \Omega(\frac1{\varepsilon^2}\log(\frac n\delta))k=Ω(ε21​log(δn​)). Let Π∈Rk×d\boldsymbol{\Pi}\in\mathbb{R}^{k \times d}Π∈Rk×d be a matrix of iid N(0,1k)\mathcal{N}(0,\frac1k)N(0,k1​) entries. Then, with probability 1−δ1-\delta1−δ, for all pairs i,ji,ji,j we have

(1−ε)∥xi−xj∥2≤∥Πxi−Πxj∥2≤(1+ε)∥xi−xj∥2 (1-\varepsilon) \|\mathbf{x}_i - \mathbf{x}_j\|_2 \leq \|\boldsymbol{\Pi}\mathbf{x}_i - \boldsymbol{\Pi}\mathbf{x}_j\|_2 \leq (1+\varepsilon) \|\mathbf{x}_i - \mathbf{x}_j\|_2 (1−ε)∥xi​−xj​∥2​≤∥Πxi​−Πxj​∥2​≤(1+ε)∥xi​−xj​∥2​

Proof. For all pairs i,ji,ji,j, consider vi,j := xi−xj\mathbf{v}_{i,j} \;{\vcentcolon=}\; \mathbf{x}_i - \mathbf{x}_jvi,j​:=xi​−xj​, and union bound Lemma 1 to find a matrix G\boldsymbol{G}G which preserves the norms of all vi,j\mathbf{v}_{i,j}vi,j​. This involves union bounding over (n2)=O(n2)\binom{n}{2} = O(n^2)(2n​)=O(n2) vectors, which causes log⁡(nδ)\log(\frac n\delta)log(δn​) to appear in the requirement on kkk. Since Π=1kG\boldsymbol{\Pi} = \frac1{\sqrt k} \boldsymbol{G}Π=k​1​G, we have

1−ε∥xi−xj∥2≤∥Πxi−Πxj∥2≤1+ε∥xi−xj∥2 \sqrt{1-\varepsilon} \|\mathbf{x}_i - \mathbf{x}_j\|_2 \leq \|\boldsymbol{\Pi}\mathbf{x}_i - \boldsymbol{\Pi}\mathbf{x}_j\|_2 \leq \sqrt{1+\varepsilon} \|\mathbf{x}_i - \mathbf{x}_j\|_2 1−ε​∥xi​−xj​∥2​≤∥Πxi​−Πxj​∥2​≤1+ε​∥xi​−xj​∥2​

We then note that 1−ε>1−ε\sqrt{1-\varepsilon} > 1-\varepsilon1−ε​>1−ε and 1+ε<1+ε\sqrt{1+\varepsilon} < 1+\varepsilon1+ε​<1+ε (see this on Desmos), which completes the proof.

See Also

The proofs above are ubiquitous, but can be found for example in Wainwright (2015) and Musco (2018).

Here's some important papers in the area of JL:

  • Johnson Lindenstrauss (1984) is the original paper of Johnson and Lindenstrauss.

  • Musco (2018) has a nice short proof, which this page basically copies.

  • Larsen Nelson (2017) shows that no other mapping, even possibly nonlinear mapping, can outperform the rate of k=Ω(1ε2log⁡(n))k = \Omega(\frac{1}{\varepsilon^2} \log(n))k=Ω(ε21​log(n)).

  • Ailon Chazelle (2009) shows the Fast-JL Embedding, which samples columns from Fourier or Hadamard matrices, allowing for faster computation of the matrix-vector product Πx\boldsymbol{\Pi}\mathbf{x}Πx.

  • Kane Nelson (2014) gives the sparsest known embedding matrix Π\boldsymbol{\Pi}Π.

  • Nelson Nguyễn (2013) lower bounds the neccessary sparsity.

  • Let me know if anything is missing
